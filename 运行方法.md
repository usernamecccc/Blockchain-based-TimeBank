
1. 启动 MySQL
服务开起来即可，账号密码要和 application.properties 里 spring.datasource.* 一致。

2. （仅当要链上功能）开本地区块链

终端 A（一直开着）：
cd D:\Blockchain-based-TimeBank\TimeCoinSystem
npx hardhat node
看到类似 Started HTTP ... at http://127.0.0.1:8545/ 再继续。

终端 B（节点起来后再执行）：
cd D:\Blockchain-based-TimeBank\TimeCoinSystem
npx hardhat deploy --network localhost
若控制台打印的 合约地址 和 application.properties 里的 blockchain.contract-address 不一致，先改配置文件里的地址，再启动后端。

3. 启动后端

cd D:\Blockchain-based-TimeBank\backend\timecoin-web
mvn spring-boot:run
或在 IDEA 里运行 TimecoinWebApplication。成功后一般是 http://localhost:8080。
netstat -ano | findstr ":8080"
taskkill /PID ** /F

4. 启动前端

cd D:\Blockchain-based-TimeBank\vue-system\pc
npm run serve
浏览器打开 http://localhost:7000，用 管理员（role = 3） 登录后可进时间币相关菜单。

5.用户端：

在 PowerShell：

cd D:\Blockchain-based-TimeBank\vue-system\phone
npm install
npm run serve
开发服务器端口在 vue.config.js 里写的是 7001，一般浏览器打开：

http://localhost:7001
