## 三、测试内容

### 3.1 认证与会话


| 测试项      | 操作                       | 结果          |
| -------- | ------------------------ | ----------- |
| 未登录访问接口  | 不带 token 访问 `/users/old` | 返回未登录，正常    |
| 错误 token | 修改 token 后重放请求           | 返回未登录，正常    |
| 登录功能     | 管理员/老人/志愿者分别登录           | 可正常获取 token |

| 密码存储     | 注册后查看数据库                 | 密码明文保存，存在风险 |


### 3.2 权限与越权


| 测试项       | 操作                                   | 结果         |
| --------- | ------------------------------------ | ---------- |
| 志愿者访问管理接口 | 用志愿者 token 访问 `/administrator/users` | 被拒绝，正常     |
| 老人访问志愿者接口 | 用老人 token 访问志愿者相关接口                  | 被拒绝，正常     |
| 未登录访问统计   | 无 token 访问 `/dashboard/stats`        | 可直接访问，存在风险 |
| 修改他人数据    | 在请求中修改活动 id 等参数                      | 暂未发现明显越权   |


### 3.3 输入与前端


| 测试项      | 操作                                 | 结果                |

| Token 存储 | 查看浏览器 localStorage                 | token 保存在本地，有一定风险 |
